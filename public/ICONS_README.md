# Generate PWA Icons

This folder contains PWA icons. If you want to use custom PNG icons instead of SVG:

1. Create 192x192 and 512x512 PNG images
2. Name them `icon-192.png` and `icon-512.png`
3. Place them in the `public` folder

You can use tools like:

- https://realfavicongenerator.net/
- https://favicon.io/
- Figma/Photoshop/GIMP

For now, SVG icons are being used as fallback.
