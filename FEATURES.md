# 🎯 Nidius Suite - Features Overview

## 🆕 Nuevas Características v2.0

### 📱 Progressive Web App (PWA)

- **Instalable**: Funciona como app nativa en cualquier dispositivo
- **Offline-First**: Service Worker para uso sin conexión
- **App Shortcuts**: Acceso rápido a tablero y optimizador
- **Iconos personalizados**: SVG escalables para cualquier resolución

### 🌓 Sistema de Temas

- **Modo Oscuro**: Tema por defecto con colores suaves
- **Modo Claro**: Tema alternativo con alta legibilidad
- **Transiciones suaves**: Cambios animados entre temas
- **Persistencia**: Guarda tu preferencia automáticamente
- **CSS Variables**: Fácil personalización de colores

### 🌍 Internacionalización (i18n)

- **Español**: Idioma por defecto
- **Inglés**: Traducción completa
- **Cambio instantáneo**: Sin recargar página
- **Extensible**: Fácil agregar más idiomas
- **Context API**: Acceso global a traducciones

### 💾 Gestión de Datos

- **Exportar**: Descarga tu progreso en JSON
- **Importar**: Restaura desde respaldo
- **LocalStorage**: Persistencia automática
- **Sincronización**: Eventos para mantener UI actualizada

### ♿ Accesibilidad

- **ARIA Labels**: Todos los controles interactivos
- **Navegación por teclado**: Soporte completo
- **Screen readers**: Compatible con lectores de pantalla
- **Contraste**: Cumple con WCAG 2.1 AA
- **Focus visible**: Indicadores claros de navegación

### 🔒 Seguridad y Privacidad

- **Sin backend propio**: Tus datos en TU navegador
- **API Keys locales**: Nunca pasan por nuestros servidores
- **Zero tracking**: Sin analytics de terceros
- **Open Source**: Código auditable

### 🎨 UI/UX

- **Glassmorphism**: Diseño moderno con efectos de vidrio
- **Responsive**: Funciona en móvil, tablet y desktop
- **Animaciones**: Transiciones suaves y feedback visual
- **Loading states**: Indicadores de progreso claros
- **Error handling**: Mensajes de error específicos

### 🧪 Testing (Setup Incluido)

- **Vitest**: Framework de testing moderno
- **React Testing Library**: Best practices para React
- **Coverage reports**: Análisis de cobertura
- **Ejemplos**: Test de App component incluido

## 📊 Características Core

### 📋 Tablero Kanban

- Drag & drop entre columnas
- 5 estados: Por Aplicar, Aplicado, Entrevista, Oferta, Rechazado
- Contador de trabajos por columna
- Scraper integrado para RemoteOK
- Generador de cartas de presentación con IA

### 🧠 Optimizador de CV

- Upload de PDF con extracción automática
- Análisis con Llama 3 70B
- Puntuación 0-100 con feedback
- Reescritura optimizada para ATS
- Adaptación a ofertas específicas

### 🎤 Simulador de Entrevistas

- Roleplay con reclutador virtual
- Preguntas técnicas y de comportamiento
- Feedback en tiempo real
- Preparación específica por empresa

### 📊 Analytics

- Tasa de conversión por etapa
- Embudo de contratación visual
- Actividad en el tiempo
- Gráficos con Recharts

## 🛠️ Stack Técnico

### Frontend

- React 19 (última versión)
- Vite 7 (build tool)
- Lucide React (iconos)
- Recharts (gráficos)
- React Markdown (renderizado)
- PDF.js (lectura de PDFs)

### Backend

- Python 3.10+
- FastAPI (API REST)
- Uvicorn (ASGI server)
- Requests (HTTP client)

### DevOps

- Docker (containerización)
- Google Cloud Run (deployment)
- Service Worker (PWA)
- GitHub Actions ready

### AI/ML

- Groq Cloud API
- Llama 3 70B (modelo de lenguaje)
- JSON mode para respuestas estructuradas
